Retrobase Home Page

About

Retrobase is a database of retrotransposon DNA sequences acquired from UCSC Table Browser and proteins translated in silico from these sequences. All protein name labels are predicted using psi-BLAST. More information on acquiring data available here. This resource is not currently intended to provide deep and detailed data of retroviral proteins. Links are provided to Uniprot and NCBI pages that serve that purpose. Instead, it serves to make accessible and queryable the many variants of retrotransposon DNA and protein sequences.

Query Tool

Querying Retrobase allows exact matching of query sequences (whole or partial) to sequences in the database. Currently BLAST algorithms are not available.
