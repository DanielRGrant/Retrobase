Retrobase Home Page

About

Retrobase is a database of retrotransposon DNA sequences acquired from UCSC Table Browser and proteins translated in silico from these sequences. All protein name labels are predicted using psi-BLAST. More information on prediction method available here.

Query Database

Querying Retrobase allows exact matching of query sequences (whole or partial) to sequences in the database. Currently BLAST algorithms are not available.

Tissue Expression Atlas

The Tissue Expression Atlas is tool for community sourcing data on the tissue specificity of expression of retrotransposon protein sequence variants.
